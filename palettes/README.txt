To change palettes, replace these .pal files with ones generated from https://bisqwit.iki.fi/utils/nespalette.php.

1. Select desired Hue, Saturation, Contrast, Brightness, and Gamma values.
2. Check the "Include de-emphasis versions in the .pal file" box.
3. Download file to this directory, naming it either ntsc_normal.pal or ntsc_grayscale.pal.
